# TikTalk

TikTalk — это веб-приложение социальной сети, разработанное с использованием Angular (Angular CLI v18.2.9).

## Функциональность

- Авторизация пользователей
- Создание постов и комментариев
- Поиск пользователей
- Редактирование профиля и загрузка аватара

##  Стек технологий

- Angular
- TypeScript
- RxJS
- Angular Signals
